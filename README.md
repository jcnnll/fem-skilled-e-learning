# Skilled E-Learning Landing Page

## Overview

This project is a **practice build** to refresh my front-end development skills using **HTML and CSS only**.
The goal was to implement a **fully responsive landing page** as close to the design as possible.

**Stack:** HTML, CSS, Vite for HMR

**Focus Areas:** Responsive design, layout implementation, accessibility and CSS interactivity

---

## Project Goals

- Implement the **Skilled E-Learning Landing Page** design faithfully using only HTML and CSS.
- Build a **responsive layout** for the mobile, tablet and desktop screen sizes
- Ensure **accessible and semantic HTML** structure.
- Maintain a **clean, well-organized project structure**.
- Deliver a deployable static site..

---

## Implementation Notes

### Layout and Styling

- Used **CSS Grid and Flexbox** for multi-column sections and responsive layouts.
- Followed a **mobile-first approach**, adjusting breakpoints for tablet and desktop.
- Included hover and focus states for interactive elements using CSS only.

### Accessibility

- Semantic HTML5 tags (`header`, `main`, `section`, `footer`) used throughout.
- Focus states tested for keyboard navigation.
- Ensured color contrast meets accessibility standards.
